# Koekoek Social Recruiting — Website

Website voor Koekoek Social Recruiting, een bureau gespecialiseerd in recruitment marketing via Meta (Facebook & Instagram).

## Stack

Plain HTML + CSS + vanilla JS. Geen framework, geen build-stap. Direct deploybaar op Netlify.

Voor styling: gebruik Tailwind via CDN (`<script src="https://cdn.tailwindcss.com"></script>`).

## Design

De `design-taste-frontend` skill is actief. Gebruik de principes voor alle UI-beslissingen:
- Typografie: Geist, Outfit of Cabinet Grotesk (via Google Fonts) — nooit Inter
- Geen emoji's als iconen — gebruik SVG iconen
- Geen pure zwart (`#000000`) — gebruik `#0a0a0a` of donkergrijs
- Geen generieke 3-koloms card layouts
- Animaties via CSS transforms en opacity, niet via positie
- Mobile-first
